二.基本操作

1. 完成代码 
2. 打开文件所在位置（右键代码区）-在目录栏输入进入cmd窗口
3. 输入 javac 文件名.java  生成class文件
4. 输入 java 类名 运行主类

三.注意事项

​	1.注意主类名与文件名一致

​	2.运行的是主类 java 类名 不要加后缀名
